Hey, I am Sophia, a Statistician/Data Scientist with a background in Mathematical Finance, and really curious about scrutinizing data to get a different perspective on real-world problems.
I started this blog project when a broken foot forced me to take a break from the outdoors. It is intended as a platform for sharing some of my writing and data works from everyday topics such as climbing, exercise, co2-consumption, and others - hopefully. If you have any comments, feedback, ideas, or issues, please drop a message!

My work and thinking is focused on the goal of contributing to a society that uses its ressources responsiblty. Inspired by the research from 80,000 hours, I hope to think that while there many pressing problems in the world, there is a way we can make a difference. Be it Climate Change, Global health, Power-seeking AI systems, Engineered pandemics or others. For more information, please check out <a href="https://80000hours.org/problem-profiles/" target="_blank">the 80,000 hour problem profiles</a> .

The blog is hosted on github (<a href="https://github.com/sophchl/sophchl.github.io" target="_blank">source code</a>), using the Jekyll theme Chirpy (which can be found <a href="https://github.com/cotes2020/jekyll-theme-chirpy" target="_blank">here</a>).
